# 🗺️ Спорт Тех - Яндекс Карты Клон

## 🌟 Особенности

### AI-генерированные аватарки
Используется DiceBear Personas API для создания уникальных аватарок на основе имени пользователя:
```typescript
avatar: "https://api.dicebear.com/7.x/personas/svg?seed=Maksim-Lutsai"
```

Преимущества:
- ✅ Стабильная генерация (одно имя = один аватар)
- ✅ Современный геометрический стиль
- ✅ Быстрая загрузка (SVG формат)
- ✅ Без ограничений API

### Реалистичный URL
Структура URL полностью соответствует Яндекс.Картам:
```
/maps/240/yekaterinburg/house/nachdiva_vasilyeva_34/YEAYdA5oTEQHQFtpfxlxcnhgbQ==/
?from=mapframe&ll=60.544249%2C56.824538&pt=60.544249%2C56.824538&source=mapframe&utm_source=mapframe&z=17.13
```
